# BC厨房

面向情侣/伴侣的治愈系双人食材管理与协同做饭软件。

## 功能模块

### 电子冰箱
- 食材手动/快速录入（生鲜、冷冻、调料、零食饮料）
- 保质期可视化（3天临期、已过期提醒）
- 每日 7:00 临期与过期食材推送提醒
- 智能补货清单 + 一键复制文本

### 今日菜单
- 自定义菜谱库（难度、耗时、步骤、食材关联）
- 今日食谱（早/午/晚/夜宵）
- 确认开饭自动扣减库存
- 库存不足柔和提示

### 留言板
- 实时数据同步（3秒轮询）
- 双人留言板 + 萌系表情

## 快速开始

```bash
# 安装依赖
npm install

# 初始化数据库并填充示例数据
npm run db:setup

# 启动开发服务器
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000)

**体验双人同步**：打开两个浏览器窗口，一个设置名字为「老公」，另一个为「老婆」，操作会实时同步。

## 技术栈

- Next.js 16 + React 19 + TypeScript
- Tailwind CSS 4
- Prisma + SQLite
- date-fns / lucide-react
